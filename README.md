Intent-Based Nearby Place Recommendation Web App

A responsive frontend web application that recommends nearby places based on user intent such as Work, Date, Quick Bite, or Budget.
Built using vanilla JavaScript, this project demonstrates real-world decision mapping, dynamic UI updates, and scalable architecture designed for future integration with location and places APIs.
